# GitFrame

GitFrame is a developer analytics dashboard that transforms your GitHub data into a clean, visual interface.

## Features
- 📊 Repository statistics (stars, forks, followers)
- 🔥 Commit activity heatmap
- 🧠 Language usage breakdown
- 📈 PR & issue timeline
- ⚡ GitHub API rate limit tracking
- 🪄 README card generator (copyable markdown)

## Tech Stack
- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- TanStack Query
- Zustand
- Recharts
- Jest + React Testing Library
